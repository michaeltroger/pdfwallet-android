# PDF Wallet
Cartera PDF simple para tus archivos PDF más importantes

Una aplicación completamente de código abierto. Esta aplicación es una cartera de PDFs fácil de usar. Esto significa que se encarga de almacenar y mostrar tus certificados PDF más importantes. Incluye una cómoda visualización de códigos de barras a pantalla completa, si el PDF contiene uno.

## Características
- **Código Abierto y Offline**: Valora tu privacidad. Sin anuncios, sin seguimiento.
- **Visualización Cómoda**: Visualización de códigos de barras a pantalla completa para acceso rápido.
- **Soporte para Muchos Códigos de Barras**:
  - 2D: QR, Aztec, DataMatrix, PDF417
  - 1D: UPC-A, UPC-E, EAN-8, EAN-13, CODE-39, CODE-93, CODE-128, ITF, Codabar
- **Más**:
  - Detectar y extraer códigos de barras en PDFs
  - (Opcional) Autenticación por huella dactilar
  - Ordenar y renombrar PDFs
  - Compartir PDFs desde/hacia otras apps
  - Soporte de Modo Oscuro
  - Soporte para PDFs protegidos con contraseña
