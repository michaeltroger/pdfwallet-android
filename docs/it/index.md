# PDF Wallet
Portafoglio PDF semplice per i tuoi PDF più importanti

Un’app completamente open source. Questa app è un semplice Portafoglio PDF. Pensato per memorizzare e visualizzare i tuoi certificati PDF più importanti. Include una comoda visualizzazione a schermo intero dei codici a barre, se presenti nel PDF.

## Funzionalità
- **Open Source e Offline**: Tutela la tua privacy. Nessuna pubblicità, nessun tracciamento.
- **Visualizzazione Comoda**: Visualizzazione a schermo intero dei codici a barre per un accesso rapido.
- **Supporto per molti codici a barre**:
  - 2D: QR, Aztec, DataMatrix, PDF417
  - 1D: UPC-A, UPC-E, EAN-8, EAN-13, CODE-39, CODE-93, CODE-128, ITF, Codabar
- **Altro**:
  - Rileva ed estrae i codici a barre dai PDF
  - (Opzionale) Autenticazione tramite impronta digitale
  - I PDF possono essere ordinati e rinominati individualmente
  - Condividi PDF da/verso altre app
  - Supporto per Modalità Scura
  - Supporto per PDF protetti da password
